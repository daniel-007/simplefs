
# Simplefs

按照 Facebook Haystack 论文用 Golang 实现的文件存储系统, 仅作学习用.

> 论文原文: [facebook haystack](http://www.usenix.org/event/osdi10/tech/full_papers/Beaver.pdf)

## 组件

* api. 用来测试和展示效果的 HTTP API
* core. 对 Haystack 中 `needle`, `directory`, `store` 的实现
* main. `Main` 函数
* utils. 其它功能

## 参考
